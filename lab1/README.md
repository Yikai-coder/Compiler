## 2 使用方法

在命令行中运行.\\release\\LexicalAnalyzer.exe即可；

默认使用.\\input_code.txt作为输入文件，.\\output.txt作为输出文件；

如果在运行时加上一个参数，则使用该参数对应的路径作为输入文件，输出文件仍为.\\output.txt;

如果运行时加上两个参数，则使用第一个参数路径作为输入文件，第二个参数路径为输出文件；
